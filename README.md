### Setup

Clone this repository to your desired folder:

```bash

```

### Install

To install all dependencies, run the following command:

```bash
npm install
```

### Run Tests

To run tests, use the following commands:

- To track CSS linter errors, run:

```bash
npx stylelint "**/*.{css,scss}"
```
